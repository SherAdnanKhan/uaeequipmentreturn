UAE Equipment Rental WordPress Deployment Notes

Local deploy files:
- deploy/uae-equipment-rental-theme.zip
- deploy/uae-equipment-rental-production.sql
- deploy/uae-equipment-rental-seo-growth.sql
- deploy/uae-equipment-rental-local-test.sql
- deploy/uae-equipment-rental-deploy-package.zip

Target domain:
- https://uaeequipmentrental.ae

Important:
- Your cPanel already has WordPress installed in public_html.
- Do not upload/replace wp-admin, wp-includes, or the whole local wordpress folder.
- Upload only the custom theme zip, then import the database/content.
- The deploy package zip is only a convenience bundle. If you use it, extract it first and then use the theme zip and SQL file inside it.
- Do not upload uae-equipment-rental-deploy-package.zip directly as a WordPress theme.

1. Backup the current live WordPress
- In cPanel, open phpMyAdmin.
- Select the WordPress database used by public_html/wp-config.php.
- Export a backup before importing the new SQL.

2. Upload the theme
- In cPanel File Manager, open public_html/wp-content/themes.
- Upload uae-equipment-rental-theme.zip.
- Extract it there.
- It should create this folder:
  public_html/wp-content/themes/dubai-bobcat-rental

3. Install the required plugins on live WordPress
- Polylang
- Fluent Forms
- Yoast SEO
- CookieYes / GDPR Cookie Consent
- Redirection
- Safe SVG
- Regenerate Thumbnails
- UpdraftPlus

Optional after launch:
- LiteSpeed Cache, if the hosting server supports LiteSpeed.
- Wordfence, after the site is stable.

4. Import the database/content
- In phpMyAdmin, select the live WordPress database.
- Import deploy/uae-equipment-rental-seo-growth.sql for the latest build with local SEO pages and 30 guide articles.
- deploy/uae-equipment-rental-production.sql is also updated to the same latest content, but use the SEO growth file so the purpose is clear.
- deploy/uae-equipment-rental-local-test.sql is only for local testing because it keeps localhost URLs.
- This will replace the fresh WordPress content with the built site content.
- The imported local admin may become:
  username: admin
  password: Admin@123456
- Change this password immediately after login.

5. Activate and refresh WordPress settings
- Login at https://uaeequipmentrental.ae/wp-admin.
- Go to Appearance > Themes and activate UAE Equipment Rental.
- Go to Settings > Permalinks and click Save Changes.
- Check Settings > General:
  WordPress Address: https://uaeequipmentrental.ae
  Site Address: https://uaeequipmentrental.ae

6. Check the site
- Home page loads correctly.
- These SEO landing pages load:
  /bobcat-rental-dubai/
  /bobcat-rental-sharjah/
  /bobcat-rental-abu-dhabi/
  /bobcat-rental-ajman/
  /bobcat-rental-fujairah/
  /bobcat-rental-rak/
  /skid-steer-loader-rental-uae/
  /cat-226b-rental-uae/
- Arabic home page opens from /ar/ and redirects to the Arabic home URL.
- Language switcher changes between EN and AR.
- Main menu links work.
- Contact form page loads at /contact/.
- Old /contact-and-quote-request/ URL redirects to /contact/.
- Old /services/bobcat-rental-dubai/ redirects to /bobcat-rental-dubai/.
- Old /machines/cat-226b-skid-steer-loader/ redirects to /cat-226b-rental-uae/.
- Arabic contact page shows the Arabic quote form.
- WhatsApp links open +971 54 738 8695.
- Blog and guide pages load.
- English and Arabic blog/guide pages load. The latest SEO build adds 30 English guide articles and 30 Arabic versions.
- Cookie banner can be accepted/closed.
- Mobile view looks correct.

7. cPanel file notes
- public_html/index.php must remain.
- If a real public_html/index.html exists, rename it because it can override WordPress.
- public_html/index.html_ is not a problem.
- Old folders like css, fonts, images, and js can stay temporarily; remove later only after confirming they are not used.

8. After launch
- Enable SSL / Force HTTPS in cPanel or WordPress.
- Install/activate LiteSpeed Cache if supported.
- Create an UpdraftPlus backup.
- Submit sitemap in Google Search Console:
  https://uaeequipmentrental.ae/sitemap_index.xml
  https://uaeequipmentrental.ae/sitemap.xml
- In Google Search Console, inspect both English and Arabic URLs after launch.
- Create or optimize Google Business Profile with the same name, phone, website, headquarters, 24/7 hours and service areas.
- Add real customer photos and ask real customers for reviews mentioning the actual service/location.
- Add matching NAP citations on Bing Places, Facebook, Instagram, LinkedIn and UAE construction/equipment directories.
